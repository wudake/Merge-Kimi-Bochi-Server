## 六、已确认需求决策（13项）

| # | 决策项 | 确认结果 |
|---|--------|----------|
| 1 | 多平台分发 | 仅记录关联账号，运营手动发布 |
| 2 | NAS 访问 | 点击自动打开 smb://，降级为复制路径 |
| 3 | 报表导出 | Excel + PDF 两者都要 |
| 4 | 审批流程 | 不需要审批 |
| 5 | 部署环境 | 本地服务器 / 内网部署 |
| 6 | 域名证书 | 直接使用 IP，暂不配 HTTPS |
| 7 | 备份策略 | 每天增量 + 每周全量，保留 30 天 |
| 8 | 账号对接 | 系统独立管理用户 |
| 9 | UI 设计 | 无设计师，Ant Design 默认风格 |
| 10 | 图表风格 | 简洁数字卡片 + 折线/柱状图 |
| 11 | 平板适配 | 低优先级，暂不处理 |
| 12 | 历史数据导入 | 需要导入账号和选题库 |
| 13 | 数据现状 | 分散在多个 Excel 中，格式不统一 |

## 十、快速启动命令

```bash
# 启动数据库
docker-compose -f docker/docker-compose.yml up -d

# 安装依赖
pnpm install

# 启动后端（http://localhost:3000）
pnpm dev:api

# 启动前端（http://localhost:5173）
pnpm dev:web

# 数据库迁移
pnpm db:migrate

# 数据库种子（创建 admin 账号）
pnpm db:seed
```
